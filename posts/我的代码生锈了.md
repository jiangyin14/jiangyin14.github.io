# 我的代码生锈了

对，是的，那个该死的红色螃蟹（

我决定从今往后，在我的所有新项目中，尽可能采用 Rust 语言进行编写（除非这玩意需要用 torch 或者 panda 或者这是一个学术研究或者这是一个AI项目🤣👉）

OK, But Why?

这是一个命题，那就是对于普通开发者来说，用 Rust 代替 Python 是必要的吗？

那么在接下来的10天里，我会尝试去探索这个问题的答案，也希望可以给大家的迁移带来一些思考🤔

对于企业来说，用 Rust 构建业务 Infra 仿佛早就不稀奇了，可是个人开发者中，大家都对这玩意感兴趣，真的抛弃了 Python 而去用的还真没那么多（？）

反正我先试试😋
